### テストの準備
- `cargo compete init atcoder` をルートで実行し、compete.toml などの必要ファイルを作成
- `cargo compete new abc189` でテストごとにフォルダを作成. この時テストケースやフォルダを自動作成.
- 作成したフォルダでテストを実行. `cargo compete test a`
- `cargo compete submit a` で提出
